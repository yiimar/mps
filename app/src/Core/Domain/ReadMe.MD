# Domain

### Уровень предметной области (или уровень модели)

Отвечает за представление концепций бизнеса, информации о деловой ситуации и бизнес-правилах. Состояние, отражающее бизнес-ситуацию, здесь контролируется и используется, хотя технические детали его хранения делегируются инфраструктуре. Этот уровень является сердцем бизнес-программного обеспечения.
